Método extra - HTML - Aula de tags e estrutura

A tag (P - < p>) é uma tag que retorna parágrafos, e possui a tag de abertura e fechamento.

A tag (Img - < img>) é uma tag que adiciona imagens na tela, e são tags "vazias pois nao tem "abre e fecha"

Quando utilizamos uma tag, como por exemplo (< a>), dizemos que este é um elemento aninhado, pois tem um elemento dentro de outro

ID é utilizado para identificar um elemento unico na página

Class é utilizado para identificar um grupo de elementos

utilizamos < ! -- para comentar linhas de código no HTML

Semântica é o estudo do significado e interpretação das palavras. Serve para deixar explicito para a máquina qual o conteudo que sera acessado.
